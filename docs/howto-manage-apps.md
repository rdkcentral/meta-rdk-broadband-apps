# How-to: Manage Containerized Apps

This guide provides walkthroughs for running and managing containerized applications on RDK broadband devices using either DAC (Dobby and DSM utilities from [meta-rdk](https://github.com/rdkcentral/meta-rdk)) or LCM (Lifecycle Management from [prplLCM](https://gitlab.com/prpl-foundation/prplrdkb/metalayers)).

!!! note "Documentation In Progress"
    This page is currently under development. Detailed tutorials and examples will be added in future updates.

**Reference Hardware**: All examples in this guide will use the **Banana Pi BPI-R4** as the target device.

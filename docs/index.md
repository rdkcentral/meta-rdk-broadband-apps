# Getting Started

This wiki provides a guide for RDK-Broadband users to:

1. **Get** your RDK build ready for Apps by **[Installing the Toolkit &rarr;](install/001_index.md)**
2. **Build** standards-compliant, cross-platform **[Apps &rarr;](build/001_index.md)**
3. **Deploy** Apps to in-field devices and manage the **[App Lifecycle &rarr;](deploy/001_index.md)**
